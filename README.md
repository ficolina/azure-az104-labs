# Azure AZ-104 Labs

Hands-on Azure infrastructure labs built with:
- Terraform
- Azure CLI
- Azure Portal

## Goal

Prepare for Microsoft AZ-104 certification by building real Azure infrastructure.

## Structure

- 01-resource-groups
- 02-networking
- 03-compute
- 04-storage
- 05-identity
- 06-monitoring-backup

Each folder represents a practical implementation of an AZ-104 exam domain.
